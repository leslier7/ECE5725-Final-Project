Introduction

This project demonstrates a dual-mode human–robot interaction system built using a custom IMU-based gesture glove and a Raspberry Pi controlled robot car. The system supports real-time control and a game interaction mode.

Project Objective

The objective of this project is to design a robust, intuitive, and responsive gesture-controlled interface for robotics and interactive gaming. Our goals include accurate IMU processing, wireless control, and two fully functioning application modes.
